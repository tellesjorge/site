# Fotos do Perfil Executivo

Adicione a sua foto profissional nesta pasta com as seguintes especificações para que ela carregue automaticamente no site:

1. **Nome do Arquivo**: `jorge-telles-profile.jpg`
2. **Localização**: `src/assets/profile/` (esta pasta)
3. **Formato recomendado**: `.jpg` (retrato executivo com fundo neutro ou desfocado, enquadramento do busto)

Enquanto a foto não for colocada aqui, o site exibirá automaticamente um fallback de avatar premium com as iniciais "JT" em gradiente azul Apple.
